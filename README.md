# PeripheralOS

PeripheralOS is a governance-first platform for programmable peripherals.

The repository is split into:

- `platform/`: standards, RFCs, ADRs, schemas, templates
- execution surfaces: runtime, sdk, plugins, transports
- `implementations/`: concrete device and runtime implementations

The platform is defined by specifications and approved decisions.
Implementations conform to the platform and do not redefine it.
