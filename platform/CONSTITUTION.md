# PeripheralOS Constitution

## Mission

PeripheralOS exists to define a universal execution platform for programmable peripherals.

## Core Values

- Stability over novelty.
- Contracts over implementations.
- Plugins over modifications.
- Composition over coupling.
- Version everything.
- Test everything.
- Document everything.

## Immutable Rules

- The Core must remain device-agnostic.
- The ABI must remain backwards compatible whenever possible.
- Applications never define the architecture.
- Devices never redefine the platform.
- Implementations conform to the platform.
- Implementations do not create new contracts.

## Foundation Freeze

No implementation may redefine the platform.

The platform is defined exclusively by:

1. Constitution
2. Specifications
3. Approved RFCs

Implementations conform to the platform.
They never redefine it.
